Hello, my name is Liana. In spring 2014, I programmed VisuaLisPy as my Hackbright Academy project.

I hold a certain aversion to black boxes, always wanting to open them up and take a peek inside. Since I began programming, I wanted to know how a computer actually understands these languages. Thus, I chose to build an interpreter for the sake of learning some inner workings of language processing.

However, as Hackbright instructor Liz said quite accurately of me, “You really like the abstract, but you get frustrated if you can’t represent it visually.” How do I share my project with others in a visual format? And so, I set off to build VisuaLisPy -- a web-based language interpretation visualizer. Working off of Peter Norvig's Lispy, I wrote an interpreter for Scheme (a dialect of Lisp) using Python. The interpreter file was modified to output JSON, which is passed to the browser and rendered visually using the D3.js JavaScript library.

In line with my love for open education, I hope VisuaLisPy will be helpful for those who want to better understand language interpretation.
